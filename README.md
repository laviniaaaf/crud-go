# [en] CRUD in Go

Simple CRUD with a **Go backend** and **HTML, CSS, and JavaScript frontend**.  
The application uses **Docker** containers to simplify project setup and execution.

---

## Start the container
docker compose up --build

## Access
 http://localhost:8081
